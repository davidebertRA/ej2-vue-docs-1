# Api margin in Vue Chart component

Configures the chart margins.

## Properties

### bottom `number`

Bottom margin in pixels.

Defaults to *10*

### left `number`

Left margin in pixels.

Defaults to *10*

### right `number`

Right margin in pixels.

Defaults to *10*

### top `number`

Top margin in pixels.

Defaults to *10*
